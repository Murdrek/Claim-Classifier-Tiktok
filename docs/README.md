## 📌 Featured Reports / Reportes Destacados

### 🎬 TikTok Claims & Verification Classification
* **Phase 1: Data Inspection & Governance**
  * **Focus:** Project governance, ethical data handling, and initial exploratory data analysis (EDA) for video claims classification.
  * **Key Contribution:** Established the PACE framework foundation and identified critical engagement metrics.
* **Phase 2: Predictive Modeling (Logistic Regression)**
  * **Focus:** Building, encoding, and evaluating a baseline discriminative classifier to predict account verification status using engagement features.
  * **Key Contribution:** Extracted log-odds coefficients to isolate key engagement drivers and mapped model limitations through a detailed confusion matrix analysis.

### 🎬 Clasificación de Reclamos y Verificación en TikTok
* **Fase 1: Inspección de Datos y Gobernanza**
  * **Enfoque:** Gobernanza del proyecto, manejo ético de datos e análisis exploratorio inicial (EDA) para la clasificación de reclamos en videos.
  * **Contribución Clave:** Establecimiento de la base del marco de trabajo PACE e identificación de métricas críticas de participación (*engagement*).
* **Fase 2: Modelado Predictivo (Regresión Logística)**
  * **Enfoque:** Construcción, codificación y evaluación de un clasificador discriminativo base para predecir el estado de verificación de las cuentas utilizando variables de interacción.
  * **Contribución Clave:** Extracción de coeficientes de *log-odds* para aislar los principales impulsores de interacción y mapeo de las limitaciones del modelo mediante un análisis detallado de la matriz de confusión.
